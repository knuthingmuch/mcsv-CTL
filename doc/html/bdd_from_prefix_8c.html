Creates a BDD from an array of tokens representing boolean formula and returns root node. Don't forget to increment reference count of returned bdd root node in he caller function by using Cudd_Ref().

Also takes an array of created bdd variable nodes(as an array of augmentedDdVar ptrs) and stores newly created DdNode* in the array for future use.

Be sure to allocate sufficient memory. No checks are performed whatsoever.

numTokens: #tokens in the array of tokens called tokens. seenVars: #variables seen so far by the program. New variables encountered will be added to this array. numSeenVars: Size of te seeenVars Array. This will be incremented for each new variable encountered.

Misc: PRIMED and UNPRIMED are macros defined in the main file so that primed and unprimed variables can be tagged once and for all.
